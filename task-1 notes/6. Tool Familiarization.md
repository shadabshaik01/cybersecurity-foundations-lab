# Tool Familiarization

## Objective
To introduce the primary tools used by cybersecurity professionals for network analysis, vulnerability assessment, and web application testing.

## Theory & Concepts
- **Wireshark**: A network protocol analyzer that lets you see what's happening on your network at a microscopic level.
- **Nmap**: A free and open-source utility for network discovery and security auditing.
- **Burp Suite**: A leading graphical tool for testing Web application security.
- **Netcat**: Known as the "Swiss-army knife" of networking, used for reading from and writing to network connections.

## Tools Used
- Wireshark
- Nmap
- Burp Suite Community Edition
- Netcat (nc)

## Commands & Practical Steps
- **Nmap Basic Scan**:
  ```bash
  nmap -sV 192.168.56.101
  ```
- **Netcat Banner Grabbing**:
  ```bash
  nc -v 192.168.56.101 80
  ```
- **Wireshark Filter**:
  ```text
  http.request.method == "POST"
  ```

## Practical Demonstration Summary
Successfully intercepted a login request using Burp Suite and observed the underlying HTTP headers. Scanned a target using Nmap to identify open services.
